Comments #

Comments are notes that you can leave inside your html. To write a comment, you use the <--! --> tags. This syntax tells the browser than anything inside the tags are comments and should not be rendered as valid code.

Check out this comment example:

                
                    <div class="container-flex">
                        <section class="section-javascript">
                            <--! Change to a better title later -->
                            <h2>Javascript Basics<h2>
                            <p>Despite what people may think, JavaScript is quite easy to learn!<p>
                        <section>
                    <div>

Anchors #

The <a> HTML element (or anchor element), with its href attribute, creates a hyperlink to web pages, files, email addresses, locations in the same page, or anything else a URL can address. Content within each should indicate the link's destination. If the href attribute is present, pressing the enter key while focused on the element will activate it.

A handy example of the <a> tag:

                
                    <nav>
                        <ul>
                            <li><a href="https://www.example-website.com" target="_blank">Link to an example website</a></li>
                            <li><a href="https://www.example-website-2.com" target="_blank">Link to another example website</a></li>
                        </ul>
                    </nav>
