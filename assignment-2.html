Spans #

The <span> element is a generic inline container for phrasing content, which does not inherently represent anything. It can be used to group elements for styling purposes (using the class or id attributes), or because they share attribute values, such as lang. It should be used only when no other semantic element is appropriate. <span> is very much like a <div> element, but <div> is a block-level element whereas a <span> is an inline element.

Here is an example of the <span> element:

                            
                                <p>
                                    This is <span>hey look! a span</span> a paragraph.
                                </p>

Anchors #

The <a> element (known as an anchor or a link), with its href attribute, creates a hyperlink to web pages, files, email addresses, locations in the same page, or anything else a URL can address. Content within each <a> should indicate the link's destination. If the href attribute is present, pressing the enter key while focused on the <a> element will activate it.

Here's an example of a <nav> element with multiple <a> tags: 

                            
                                <nav>
                                    <ul>
                                        <li>
                                            <a href="https://www.example-website.com" target="_blank">
                                                Link to an example website
                                            </a>
                                        </li>
                                        <li>
                                            <a href="https://www.example-website-2.com" target="_blank">
                                                Link to another example website
                                            </a>
                                        </li>
                                    </ul>
                                </nav>

Comments #

Comments are notes that you can leave inside your html. To write a comment, you use the <--! --> tags. This syntax tells the browser than anything inside the tags are comments and should not be rendered as valid code.

Check out this comment example:

                            
                                <div class="container-flex">
                                    <section class="section-javascript">
                                        <--! Change to a better title later -->
                                        <h2>Javascript Basics<h2>
                                        <p>
                                            Despite what people may think, JavaScript 
                                            is quite easy to learn!
                                        <p>
                                    <section>
                                <div>

Nested Lists #

There are two kind of lists: the unordered and ordered lists. The unordered list is represented with the <ul> and the ordered list is represented with the <ol> tag. The <li> tag represents items in these lists. You can nest lists inside of each other to create to-do lists & navigational menus.

Check out this comment example:

                            
                                <ul>
                                    <li>Coffee</li>
                                    <li>Tea
                                        <ol>
                                            <li>Black Tea</li>
                                            <li>Black Tea</li>
                                        </ol>
                                    </li>
                                    <li>Milk</ol>
                                </ul>

Input Forms #

The <input> is used to create interactive controls for web-based forms in order toaccept data from the user; a wide variety of types of input data and control widgets are available, depending on the device and user agent. The <input> element is one of the most powerful and complex in all of HTML due to the sheer number of combinations of input types and attributes.

See how a simple input form is set up in HTML:

                            
                                <form>
                                    <label for="name">Enter Your Name</label><br>
                                    <input type="text" id="name" name="name">
                                </form>

Dropdowns #

The <select> element represents a control that provides a menu of options. A <select> element can have id and name attributes. The id attribute is to associate a <select> with a <label> for accessibility purposes. The name attribute represents the name of the associated data point submitted to the server. Each menu option is defined by an <option> element, and there can be multiple such elements.

Below you can see an example of the <select> element: 

                            
                                <label for="pet-select">Choose a pet</label>
                                <select name="pets" id="pet-select">
                                    <option value="">--Please choose an option--</option>
                                    <option value="cat">dog<Cat</option>
                                    <option for="dog">Dog</option>
                                </select>

Radio #

Radio buttons are used in <input> forms where a user should select only one option. To group related radio buttons, you would use the <fieldset> tag. To give the <fieldset> a title, a <legend> element is required. The checked attribute selects its radio button.

Consider the example below: 

                            
                                <form>
                                    <fieldset>
                                        <legend>Select your favorite food</legend>
                                            <div>
                                                <input type="radio" id="hamburger" name="food" value="hamburger" checked>
                                                <label for="hamburger">Hamburger</label">
                                            </div>
                                            <div>
                                                <input type="radio" id="pasta" name="food" value="pasta">
                                                <label for="pasta">Pasta</label">
                                            </div>
                                            <div>
                                                <input type="radio" id="pizza" name="food" value="pizza">
                                                <label for="pizza">Pizza</label">
                                            </div>
                                    </fieldset>
                                </form>
